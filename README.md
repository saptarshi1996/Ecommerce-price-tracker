# Ecommerce price tracker.

### This project is seperate parts which are connected using the microservices. 

react typescript frontend, typescript backend, bull queue worker, bull queue cron jobs

the worker will consist of the web scraping. 

database built using mysql and prisma.

## step 1. Create API's for login, register, verification and user details.
## step 2. Create a scraper to fetch data from ecommerce websites.
## step 3. User can add a product using an url.
## step 4. User will get periodic price updates of their products via email.
